## Setup
### 准备以下玩意儿
- Node.js
- OpenAI API Key
### 安装
1. 克隆代码:
```sh
git clone https://github.com/ioanmo226/chatgpt-web-application
```
2. 安装npm:
```sh
npm install
```
3. 在环境中添加并配置.env密钥文件
```sh
OPENAI_API_KEY=your_api_key
```
4. 启动服务
```sh
node index.js
```
